# 01. OpenTofu e Conceitos

Esta fase inicia o curso com os fundamentos do OpenTofu.

Material de apoio já ligado à trilha principal do curso:

- [7. Conceitos do OpenTofu](../../07-conceitos-do-opentofu.md)

Objetivos sugeridos para esta fase:

- entender o que é OpenTofu;
- diferenciar IaC de criação manual de infraestrutura;
- apresentar arquivos como `main.tf`, `variables.tf` e `outputs.tf`;
- explicar `init`, `plan`, `apply` e `destroy`;
- introduzir providers, resources, data sources, variables e outputs;
- explicar o conceito de state.

Estrutura:

- `assets/`: imagens, diagramas e material visual
- `exemplos/`: exemplos prontos usados durante a aula
- `exercicios/`: exercícios propostos para os alunos

Orientações da fase:

- ao final de cada etapa, o projeto do aluno deve ficar semelhante ao subdiretório de exemplo correspondente;
- instruções de credenciais e acessos aos ambientes devem ser obtidas individualmente por cada participante.

Convenção:

- se o material mencionar "terraform", considere que o curso está tratando de OpenTofu.
