# 3. Nuvem Particular (VPC)

Esta fase introduz o conceito de rede privada dentro da nuvem.

<blockquote><strong>⚡ Visão rápida:</strong> aqui começa a construção da base de rede sobre a qual os próximos recursos vão depender.</blockquote>

Em termos práticos, a `VPC` é o espaço de rede isolado onde recursos como máquinas virtuais, bancos de dados, balanceadores e serviços internos podem existir e se comunicar.

Nesta fase, a ideia é entender por que a rede é uma peça central da infraestrutura e por que muitos recursos dependem diretamente dela para funcionar corretamente.
